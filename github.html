Git

Git er et kontroll system som lar deg jobbe med kode eller andre prosjekter, og da kunne ta vare på alle endringene på en lett og oversiktelig måte, det er da mulig å gå tilbake til en hviklensom helst tidligere versjon av filen din, og gjenoppta arbeid fra den versjonen.

GitHub

Git og GitHub er ikke det samme, GitHub er bare en av måtene man kan bruke Git på. GitHub er en tjeneste som lar deg bruke git og lagre filer på netter. Github har en klient som kan lastesned på maskinen og da lar deg skrive notater på hver endring du gjør, og laste det opp på din GitHub profil. Bildet til siden for denne artikkelen er et bildet av GitHub sin klient, og viser noen endringer gjort på min github profil på denne filen.
